== Description ==
= The Ultimate Divi Carousel Lite for Divi Builder =

Elevate your website's visual appeal and user experience with Divi Carousel Lite, a versatile and feature-rich carousel plugin for Divi Builder. Create dynamic, touch-friendly, and fully responsive carousels that look stunning on all devices. With an array of customization options and premium modules, Divi Carousel Lite is the perfect solution for showcasing your content beautifully and engagingly.

= Key Features =

* Fully responsive and touch-friendly for seamless browsing on any device
* Fast, lightweight, and powerful for optimal performance
* Auto Play ON/OFF with speed control for tailored user experience
* Infinite looping to keep your content always in motion
* Show/Hide Navigation and Pagination Dots for a clean design
* Fast and active support to address your needs and concerns
* And much more!

= Enjoy 3 Premium-Like Divi Carousel Modules =

- **Image Carousel** - Craft interactive image and text carousels with a host of options
- **Logo Carousel** - Showcase your clients or products in style
- **Twitter Carousel** - Display your team's Twitter feed with various styles, texts, images, and social links

== Installation ==

= Minimum Requirements =

* WordPress 5.0 or greater
* PHP version 7.0 or greater
* MySQL version 5.0 or greater

= We recommend your host supports: =

* PHP version 7.0 or greater
* MySQL version 5.6 or greater
* WordPress Memory limit of 64 MB or greater (128 MB or higher is preferred)

= Installation =

> Note: This plugin works with Divi. Make sure you have [Divi Theme](https://www.elegantthemes.com/) installed.

1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. You can type "Wow" on your module tabs within the Divi editor and all the available modules will appear.
